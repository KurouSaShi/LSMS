## アンインストール

### PC
```C:\Users\<ユーザー名>\AppData\Local\LSMS```とインストールしたzipファイルを削除することでアンインストールができます。

### スマホ
```https://xxx.xxx.xx.xxx:xxxx/```のサイトデータとキャッシュを消去し、ホームに追加されたPWAを削除することでアンインストールができます。

### Discord共有

この機能にはDiscordBotのトークンが必要です。
所得はこちらのページからできます。

https://discord.com/developers/applications

所得したトークンを環境設定に記入し、チャンネル設定を行うことでこの機能を利用できます。

メモを選択して押すとDiscordBotを通じてメモが指定のチャンネルに投稿されます。

選択せずにやるとすべてのメモを共有します。

共有されたメモにリプライで
```
#issue <issue内容>
```
とするとissueが届きます。詳しくは[Issue](#issue)

### デバイス接続

スマホなどのデバイスでも作業できるよう、hocalhostを通じてwebappを立ち上げます。

WebappにアクセスするためのhttpURLとQRコードを表示します。

ソフトが起動しているPCのLANと同じ回線を使っている場合、どこからでもアクセスできます。(PC起動中のみ)


### Issue

メモに対し、自分や他人が意見を出し合えます。

Issueには返信できます。

DiscordからのIssueの場合、Issueを出した人にメンション付きで返信されます。

closeされたissueには返信ができません。また、再びopenにすることもできません。

### 関連付け

メモ同士を関連付けできます。
関連付けられたメモはクリック/タップするとそのメモに瞬時に飛ぶことができます。

### タグ

メモごとにタグをつけられます。
タグには名前と色を付けられ、PCではカラーコードから、スマホでは12色から選べます。

### PWAのダウンロード
PWAとしてスマホにアプリをDLします。
DLしたアプリはオフライン下でも使用できるようになり、次回ソフト起動時にソフト側がスマホと同期されるようになります。

1.デバイス接続でスマホに接続する
  
  iOSの場合は**safari**,Androidの場合は**chrome**から行ってください。

接続の際、安全でない接続・証明書のエラーが出る場合があります。エラーは無視しても大丈夫です。

2.ページの共有機能からホーム画面に追加を選択する

3.おしまい！！

オフライン起動ができない場合、serviceworkerの機能がONになっているか確認してください。

### 注意事項

ダウンロード先のブラウザ(iOSの場合Safari,Androidの場合chrome)のキャッシュやサイトデータを削除すると再DLが必要になります。

アンインストールを行う場合、PCにメモを移ったのを確認し、PCの環境設定から同期設定を**PC優先**にしてからアンインストールをお願いします。

### 基本操作

![](https://github.com/KurouSaShi/LSMS/blob/main/readme/IMG_4034.png?raw=true)

メモをタップするとそのメモを開けます。より詳しく閲覧・編集したい場合は全画面表示を押してください。

メモを開いている状態では以下の操作ができます。

- 本文の記入
- 期日の設定
- 全画面表示
- Issueの作成
- メモの削除

全画面表示中は上記に加え以下の設定もできます。

- タグの設定
- 下に(同じ断階に)メモを追加
- 子のメモ(1つ下の段階に)メモを追加
- 関連メモの追加、編集
メモをダブルタップでタイトル編集ができます。

メモを開いていなくても以下の機能は使えます
- PCとの同期
- メモの新規作成
- メモの移動
- チェックボックスのON/OFF
- メモの検索

メモを**右に**スワイプするとメモの削除が行えます
メモを**左に**スワイプすると自分の子のメモを作成できます。

![](https://github.com/KurouSaShi/LSMS/blob/main/readme/IMG_4035.png?raw=true)

ハンバーガーメニュー内は以上のようになっています。

メモ一覧をタップした後右に出る**↸**を押すとそのメモを全画面で表示できます。
![](https://github.com/KurouSaShi/LSMS/blob/main/readme/IMG_4036.png?raw=true)

Issueの管理はここで行います。
それぞれのIssueに
- 返信(追記)
- Issueのclose
- 削除ができます

並び替えもできます。

メモを引用したIssueもここに表示されます。

![](https://github.com/KurouSaShi/LSMS/blob/main/readme/IMG_4037.png?raw=true)

設定はここから行えます。

**ユーザー設定**：Issueに表示する名前を設定できます。

**表示**：
- メモの下に本文のプレビューを表示するか決めれます。
- 同期した際、下にメッセージが出る機能を停止します。

**タグ管理**：タグの設定、新規作成、削除をここから行えます。

期日通知：開発中です

LAN接続設定：接続用IPを手動瀬設定、接続を行えます。

バージョン履歴：そのまんま
