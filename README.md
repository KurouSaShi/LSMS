## できること


1.ソフト名：わかりやすいですね。

### メモの操作
2.同じ階層に追加：今選択しているメモと同じアウトライン(階層)にメモを新規作成します。
3.子に追加：今選択しているメモの下の階層にメモを新規作成します
4.チェックボックス：文字が薄くなり、期日になっても通知されなくなります。また、チェックを入れておくと何日かで自動でメモが削除されます
5.アウトライン収納：それより下のメモを非表示/表示させます
6.移動キー：ここをクリック長押しでメモを動かせます。
7.完了非表示：チェックがついているメモを非表示にします。
8.メモの詳細表示：くわしくは[本文記入](#本文記入)
9.自動保存：そのまんま
10.Issue：Issueのページに飛びます。詳しくは[Issue](#issue)
11.スマホ接続：他デバイスと接続するためのURLとQRコードが表示されます。詳しくは[スマホアプリ](#スマホアプリ)
12.設定を行います
13.Discord共有：選択したメモをDiscordに共有できます。くわしくは[Discord連携](#Discord連携)
14.デバイス共有：メモを共有しているデバイスの数を表示します。

### 本文記入
13.本文：本文を記入しますmarkdown記法が使用できます。
14.期日：期日を設定できます。近くなると通知されます。
15.選択中のメモ：そのまんま
16.検索ボックス：そのまんま

## Discord共有

この機能にはDiscordBotのトークンが必要です。
所得はこちらのページからできます。

https://discord.com/developers/applications

所得したトークンを環境設定に記入し、チャンネル設定を行うことでこの機能を利用できます。

メモを選択して押すとDiscordBotを通じてメモが指定のチャンネルに投稿されます。
選択せずにやるとすべてのメモを共有します。
共有されたメモにリプライで
```
#issue <issue内容>
```
とするとissueが届きます。詳しくはIssue](#issue)
